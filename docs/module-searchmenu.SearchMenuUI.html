Class: SearchMenuUI

searchmenu.SearchMenuUI()

new SearchMenuUI()

Search Menu UI.

Contains the "behavior" of the search bar. It submits the search query, parses the results, displays matches and filters and responds to clicks and key presses. Further resources:
